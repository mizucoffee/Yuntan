## 概要

手を振ったりヘドバンしたりする音楽ゲームのシミュレータ  
対応ファイル形式: .sus

**Yuntan was inspired by Seaurchin.**

### 使用しているもの

- Electron
- TypeScript
- Babylon.js
- Prettier
- webpack

## How to build

```sh
yarn
npm start
```
